# Student Performance Prediction using Linear Regression

## Overview
This project predicts student final exam performance using Machine Learning.

A Linear Regression model is trained on the Student Performance Dataset (UCI) to predict final grade (G3).

An interactive Streamlit dashboard allows users to input student details and view predictions.

---

## Features
- Predict student final score (G3)
- Pass/Fail prediction
- Interactive dashboard
- Data visualization charts
- Model evaluation metrics

Metrics included:
- Mean Squared Error (MSE)
- R² Score
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

---

## Dataset
Dataset: Student Performance Dataset (UCI)

Source:
Kaggle

Target variable:
G3 (Final grade)

Input features:
- studytime
- failures
- absences
- G1
- G2
